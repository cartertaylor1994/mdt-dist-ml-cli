## Overview

MDT Dist is a Python command-line tool centered on marginal-data transport distillation for 3D generation tasks. Its main purpose is to cut down the sampling steps required at inference time, with an emphasis on keeping runs efficient while remaining usable across platforms.

This project is aimed at people working with machine learning workflows, 3D generation experiments, or CLI-based distillation pipelines. Rather than a graphical application, it focuses on compact inference and automation-friendly operation.
